## ðŸ“¦ Uso de Cargo

- Crear un proyecto:
    ```bash
    cargo new <nombre_proyecto>
    ```
- Construir un proyecto:
    ```bash
    cargo build
    ```
- Construir y ejecutar en un solo paso:
    ```bash
    cargo run
    ```
- Comprobar errores sin compilar binario:
    ```bash
    cargo check
    ```
